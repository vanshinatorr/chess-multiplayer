Frequently Asked Questions

How does the real-time Evaluation Bar work? ▼

The evaluation bar uses our built-in position engine to score the board state in real time. It analyzes piece counts, pawn structures, center control, and king safety. The numerical value is rendered in standard chess centipawns where +1.0 represents a one-pawn advantage for White.

Can I invite friends to spectate my game? ▼

Yes! If two players are already active in the game room, any additional players joining using the same 6-digit room code will automatically enter as spectators. They can view the game in real time and converse in the battle chat.

Is the minimax practice bot running on the server? ▼

No, the AI is completely compiled and executed in your browser tab. This reduces network overhead, guaranteeing zero-lag thinking pauses and permitting completely offline practicing if you lose connection to the internet.
